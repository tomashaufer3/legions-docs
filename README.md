# The Fateful Adventrues of the Good Soldiers — project documentation (unlisted)

Generated documentation for a CERGE-EI master's thesis on desertion in the
Czechoslovak Legions (WWI): pipeline, variable construction, data sources, live
descriptives and QA. **Aggregates and methodology only — no individual-level
records.**

- The page is **`index.html`** (self-contained; open it directly, or via GitHub Pages).
- It carries a `noindex, nofollow` meta tag, so search engines will not list it —
  it is meant to be shared **by link**, not discovered.
- Built from a private source repository by `utils/docs/build_docs.py`; do not edit
  `index.html` by hand — regenerate and replace it.

To update: rebuild with `DOCS_NOINDEX=1 python utils/docs/build_docs.py` in the source
repo, copy the new `index.html` here, and push.
